# @aiai-go/claude Plugin for Claude Code

Install Chinese language support and 12 custom tools directly into Claude Code.

## Install

```bash
claude plugin install /path/to/plugin
```

Or manually copy to `~/.claude/plugins/aiai-go-claude/`.

## What You Get

### Chinese Commands
- `/zh` — Switch to Simplified Chinese
- `/zht` — Switch to Traditional Chinese
- `/en` — Switch back to English
- `/review-zh` — Code review in Chinese
- `/explain-zh` — Explain code in Chinese
- `/test-zh` — Generate tests in Chinese
- `/fix-zh` — Fix bugs in Chinese

### 12 MCP Tools
Project memory, code stats, git analysis, environment detection, quick notes.
